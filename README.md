# JORDAN_SITE_VIGA

This repository contains a Vite + React + TypeScript storefront app using Supabase.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```
2. Add Supabase credentials in `.env`:
   ```bash
   VITE_SUPABASE_URL=https://your-project.supabase.co
   VITE_SUPABASE_ANON_KEY=your-public-anon-key
   ```
3. Run locally:
   ```bash
   npm run dev
   ```
4. Build for production:
   ```bash
   npm run build
   ```

## Deployment

Deploy this project to any static-hosting platform that supports Vite apps, such as Vercel, Netlify, or Cloudflare Pages. Use the `npm run build` output or let the platform build the app from source.
